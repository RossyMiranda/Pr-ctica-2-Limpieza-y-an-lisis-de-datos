4.2 Normalidad y Homogeneidad de la Varianza de los Datos¶

4.2.1 Normalidad¶

Para la comprobación de que los valores que toman nuestras variables cuantitativas provienende una población distribuida normalmente, utilizaremos la prueba de normalidad de Shapiro_Wilk. Siendo la hipótesis nula que la población está distribuida normalmente, si el p-valor es menor a alfa= 0,05 (nivel de significancia) entonces la hipótesis nula es rechazada (se concluye que los datos no vienen de una distribución normal). Si el p-valor es mayor a alfa= 0,05, se concluye que no se puede rechazar dicha hipótesis.

In [315]:
from statsmodels.graphics.gofplots import qqplot
from scipy.stats import shapiro

Respuesta: Analizando los p-valor de todas las variables aplicando el test de Shapiro_Wilk se observa que se obtienen valores inferiores a alfa= 0,05 (nivel de significancia) entonces la hipótesis nula es rechazada por lo que se concluye que los datos no vienen de una distribución normal.
